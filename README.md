# react-blog-example

## Installation

Install the project dependencies:

# npm
npm install

# yarn
yarn


## Run Scripts

Run in development mode:

# npm
npm start

# yarn
yarn start

Open [http://localhost:3000](http://localhost:3000) to view in browser.

Build app for production to the `build` directory:

npm run build

# yarn
yarn build

### Testing

Run tests in watch mode:

npm test

# yarn
yarn test
